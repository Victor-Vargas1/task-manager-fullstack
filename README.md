# 🚀 TaskFlow — Gerenciador de Tarefas Fullstack

🚀 **Apresento o TaskFlow — meu gerenciador de tarefas fullstack**

Desenvolvi o **TaskFlow** com foco em produtividade e organização, permitindo gerenciar tarefas de forma simples, visual e eficiente.

💡 A proposta foi criar uma solução prática para o dia a dia, com recursos que realmente fazem diferença no controle de atividades e prazos.

---

## 🔧 Principais funcionalidades

✔️ CRUD completo de tarefas  
✔️ Definição de prazos  
✔️ Busca em tempo real  
✔️ Marcação de tarefas concluídas  
✔️ Alertas automáticos para tarefas urgentes (≤ 3 dias)  
✔️ Destaque visual para prioridades  

---

## 🧠 Diferenciais do projeto

• Organização automática (pendentes primeiro)  
• Interface moderna e responsiva  
• Experiência otimizada para produtividade  
• Layout estilo aplicativo com scroll isolado  

---

## 🛠️ Tecnologias utilizadas

🔙 **Backend:** C#, ASP.NET Core, Entity Framework, MySQL  
🎨 **Frontend:** HTML, CSS (Flexbox + Grid), JavaScript (Vanilla)

---

## 📡 Arquitetura

Aplicação fullstack com integração via API REST, garantindo comunicação eficiente entre frontend e backend e persistência de dados.
